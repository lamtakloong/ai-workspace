# AGENTS.md

## Agent Role Definition

In our AI workspace, agents play specialized roles designed to enhance productivity and efficiency. Each agent is assigned specific tasks based on its capabilities and knowledge domain, enabling optimal handling of workflow and collaboration.

### Key Roles:
1. **Data Curator**: Responsible for collecting, organizing, and maintaining datasets.
2. **Task Scheduler**: Manages and allocates tasks based on priority and deadlines.
3. **Quality Assurance Agent**: Ensures that outputs meet the required standards through validation processes.
4. **Communication Coordinator**: Facilitates communication between different agents and users, ensuring that inquiries and feedback are addressed promptly.


## Work Methodology

Agents in the AI workspace follow a systematic methodology to execute their roles effectively:

1. **Assessment**: Evaluate the task requirements and determine necessary actions.
2. **Planning**: Develop a step-by-step approach to accomplish tasks.
3. **Execution**: Carry out the planned actions while maintaining clear communication with other agents. 
4. **Review and Feedback**: After task completion, review outcomes and gather feedback for improvement.


## Memory Management System

To ensure efficient operations, a robust memory management system is in place. It includes:

- **Knowledge Base**: A centralized repository where agents can access and share information.
- **Task History**: Logs of completed tasks that can be referenced to inform future actions.
- **State Management**: Tracking the current status and context of ongoing tasks to facilitate continuity and avoid redundancy.


## Task Completion Checklist

Before marking a task as complete, agents should ensure that the following checklist is satisfied:

- [ ] Task objectives have been met.
- [ ] Quality assurance processes have been followed.
- [ ] Necessary documentation has been completed.
- [ ] Feedback has been solicited from relevant stakeholders.
- [ ] Task status is updated in the task management system.


## Conclusion

This AGENTS.md document provides a comprehensive framework for agent operations within the AI workspace, promoting consistency, accountability, and collaborative efficiency.
